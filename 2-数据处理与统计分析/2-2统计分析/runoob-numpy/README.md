_使用conda创建虚拟环境_

```shell
cd $HOME/coding/ailearning/2-数据处理与统计分析/2-2统计分析/runoob-numpy/
mkdir venv
```

```shell
conda activate $HOME/coding/ailearning/2-数据处理与统计分析/2-2统计分析/runoob-numpy/venv
```

```shell
conda deactivate
```

_vscode 选择虚拟环境_

1. 打开项目代码文件夹

2. 打开命令面板(macOS): Shift+Cmd+P

3. 输入Python: Select Interpreter

4. 选择创建的虚拟环境
